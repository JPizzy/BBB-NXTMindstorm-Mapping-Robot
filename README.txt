How to Run
--------------------------------------------------------------------
	1. On your host you can run 'make' to compile the project
	2. On the target the application can be found in your public
	   folder and then under the myApps/ directory
	3. From there navigate to the nxt-server-backup folder
	4. Run node server.js &
	5. Switch back to the project's main directory
	6. Run ./nxtMapper
	7. The website can be accessed by going to 192.168.7.2:8088
